# 📰 News App

A modern, clean, and responsive **Flutter** application that delivers real-time news articles from around the world. The application connects to **NewsAPI** to fetch publication sources and articles across multiple categories, complete with full support for Light/Dark themes, cached network images, and dynamic category filtering.

---

## ✨ Features

- **📂 Category-Based News Browsing**: Browse news articles categorized under seven main topics:
  - 💼 Business
  - 🎬 Entertainment
  - 🌍 General
  - 🏥 Health
  - 🔬 Science
  - ⚽ Sports
  - 💻 Technology
- **🏷️ Source Filtering**: Inside each category, filter articles dynamically via a scrollable list of news sources (tabs fetched in real-time).
- **🌓 Dynamic Light & Dark Mode**: Seamlessly switch between Light and Dark themes via the navigation drawer.
- **⚡ Performance-Focused Rendering**: Uses caching for network images (`cached_network_image`) to save bandwidth and ensure a smooth scrolling experience.
- **📅 Localized Date Formatting**: Article publication dates are beautifully formatted using custom extension methods (e.g., `Mon, May 19, 2026`).
- **📱 Splash Screen**: Equipped with a custom splash screen built using `flutter_native_splash`.

---

## 🛠️ Tech Stack & Dependencies

This app is built using **Flutter** and **Dart**, leveraging the following packages:

- **State Management**: [provider](https://pub.dev/packages/provider) — Manages the theme state globally.
- **Networking**: [http](https://pub.dev/packages/http) — Communication with the NewsAPI.
- **Image Caching**: [cached_network_image](https://pub.dev/packages/cached_network_image) — Efficiently downloads and caches remote images.
- **Date Formatting**: [intl](https://pub.dev/packages/intl) — Parsing and formatting dates.
- **Splash Screen**: [flutter_native_splash](https://pub.dev/packages/flutter_native_splash) — Native Android/iOS splash screens.

## 🚀 Getting Started

Follow these steps to set up the project locally:

### 1. Prerequisites

Make sure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your system.

```bash
flutter --version
```

### 2. Clone the Repository

```bash
git clone https://github.com/nda-mohamed/newss_app.git
cd newss_app
```

### 3. Fetch Dependencies

Install the packages listed in `pubspec.yaml`:

```bash
flutter pub get
```

### 4. Configure NewsAPI Key

The app connects to [newsapi.org](https://newsapi.org/). Currently, the API key is configured in `lib/api/ApiManager.dart`. 

To use your own API Key:
1. Register and get a free key from [newsapi.org](https://newsapi.org/).
2. Open `lib/api/ApiManager.dart` and update the key:
   ```dart
   static const String _apiKey = 'YOUR_NEWS_API_KEY';
   ```

### 5. Run the Application

Run the app on your connected device or emulator:

```bash
flutter run
```

## ⚙️ Architecture Details

### ApiManager (Singleton)
The API layer is encapsulated in `ApiManager` which prevents multiple instantiation. It exposes:
- `getNewsSources(String category)`: Retrieves available publications matching categories (e.g., `sports`).
- `getArticles(String sourceId)`: Retrieves list of articles published by a specific news outlet.

### ThemeProvider
Using the `ChangeNotifierProvider` pattern, the app dynamically changes themes between light and dark settings on-the-fly. The setting is triggered inside the navigation Drawer of the `CustomScaffold`.
